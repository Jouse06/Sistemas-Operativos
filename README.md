# Sistemas-Operativos

En este repositorio se encuentran los códigos y ejemplos de los sistemas operativos que se han desarrollado en la asignatura.
El que lo quiera usar o rellenar esta mas que invitado.

## Contiene los siguientes modulos.

- ***Modulo 1: Fundamentos de C***
- ***Modulo 2: Sistemas operativos y arquitectura UNIX***
- ***Modulo 3: Input/Output de bajo nivel y SYSCALLS***
- ***Modulo 4: Manejo de proccessos y forks***
- ***Modulo 5: Manejo de CPU, hilos y syncronizacion***
- ***Modulo 6: Manejo de memoria***

En cada modulo incluye preguntas de teoria y practicas.
